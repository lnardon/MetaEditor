# MetaEditor

A simple terminal application to edit your music files metadata and organize them in a structured way.

- Written in Golang
- Follows the ID3v2 standard
- MP3 files only

## Installation

1. Clone the repository

```
    git clone https://github.com/lnardon/MetaEditor.git
```

2. Build the app

```
    cd MetaEditor
    make build
```

3. Run the app

```
    ./main
```
