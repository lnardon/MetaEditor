## Installation

1. Clone the repository

```
    git clone https://github.com/lnardon/MetaEditor.git
```

2. Build the app

```
    cd MetaEditor
    make build
```

3. Run the app passing the path to the directory containing the mp3 files

```
    ./main /path/to/mp3/files
```
